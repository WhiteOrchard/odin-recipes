# LuxeEstates - Luxury Property Management

A modern web application for managing luxury residential properties. Built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- **Dashboard** - Real-time portfolio overview with KPIs, revenue charts, and occupancy metrics
- **Property Management** - Browse, filter, and manage luxury properties with image galleries
- **Tenant Portal** - Track tenants, lease agreements, and communication history
- **Financial Analytics** - Revenue tracking, expense breakdowns, and profitability analysis
- **Maintenance Tracking** - Submit, assign, and track maintenance requests with priority levels
- **Calendar** - Schedule viewings, inspections, lease renewals, and maintenance
- **Dark Mode** - Full dark/light theme toggle

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 7** for blazing fast builds
- **Tailwind CSS v4** for utility-first styling
- **React Router v6** for client-side routing
- **Recharts** for data visualization
- **Lucide React** for icons
- **date-fns** for date formatting

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## Build

```bash
npm run build
npm run preview
```
